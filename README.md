# Real Estate Property Management System

A full-stack real estate property management application.

## Project Structure

- **client/** - Next.js frontend application
- **server/** - Backend API server

## Getting Started

### Client (Frontend)

```bash
cd client
npm install
npm run dev
```

The client will be available at `http://localhost:3000`

### Server (Backend)

```bash
cd server
npm install
npm start
```

## Technologies Used

### Frontend
- Next.js 16
- React
- TypeScript
- Redux Toolkit

### Backend
- Node.js
- Express (to be configured)
